# E-Commerce | 電子商務

## 專案說明：

- 網站提供用戶自訂商店看板、分類以及產品等
- 產品照片另外儲存於Edgestore，並同步新增、刪除
- 使用Server action操作資料庫，另外建立個別API供前端商店抓取資料以及紀錄訂單資料
- 提供資料表檢視看板、分類以及產品等，並可個別編輯或複選刪除

## 開發使用到的技術：

- **語言**:  TypeScript
- **前後端**:  Next.js
- **CSS Style**: Tailwind CSS、shadcn/ui
- **Server Action**
- **資料庫**: MongoDb 
- **ORM**: Prisma
- **欄位驗證**: Zod
- **狀態管理**: Zustand
- **資料表**: TanStack Table
